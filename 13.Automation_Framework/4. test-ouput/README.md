 AUTOMATION TESTING FRAMEWORK – SHOPPERSTACK

Project Type:
Web Application Automation Testing

Application Under Test:
ShopperStack

Automation Stack:
- Language: Java
- Automation Tool: Selenium WebDriver
- Test Framework: TestNG
- Design Pattern: Page Object Model (POM)
- Build Tool: Maven
- IDE: Eclipse

Framework Structure:
- src/main/java:
  Contains utility classes and Page Object Model classes.
- src/test/java:
  Contains test scripts.
- TestNG XML files:
  Used to control and execute test cases.
- reports:
  Contains execution reports.
- test-output:
  Contains TestNG default reports.

Execution Process:
1. Test cases are written using Selenium and TestNG.
2. Test execution is controlled using TestNG XML files.
3. Tests are executed from Eclipse using:
   Right Click → Run As → TestNG Suite
4. Reports are generated after execution.

Current Scope:
- Login
- Product Search
- Add to Cart
- Wishlist
- Address Management
- Password Change
- Product Review

Status:
Automation framework is successfully implemented and tested.
